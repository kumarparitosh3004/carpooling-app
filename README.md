# Carpooling App (Hackathon Project)

This project was developed during a college hackathon organized by GDSC.

## Idea
The goal of this application is to help students from VIT share rides while traveling from Vellore to cities like Chennai or Bangalore, reducing travel costs through carpooling.

## Features
- User input form for travel details
- Collects name, mobile number, source, destination
- Option to select mode of transport
- Simple and clean user interface

## Technologies Used
- HTML
- CSS

## Status
This is a basic prototype developed during a hackathon and is being improved further.
